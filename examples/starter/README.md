# Overworld Starter

最小可玩示例,验证并演示 `@overworld/*` 全家桶的端到端组合方式。无需任何美术资产
(玩家与 NPC 均为几何体回退)。

```bash
pnpm install && pnpm build   # 仓库根目录
pnpm --filter starter dev
```

## 玩法(约 2 分钟)

1. WASD/方向键移动(或拖左下角**虚拟摇杆**,触摸/鼠标均可),Shift 跑 —— 推进任务
   「初来乍到」的行走目标,10 米解锁成就。右上角**小地图**实时显示 NPC(黄点)与
   水晶(蓝点),灯光随**昼夜循环**缓慢变化。
2. 走近向导艾拉,按 **E** 对话;接受委托后开启任务「收集能量水晶」。
3. 走近 3 颗漂浮水晶自动拾取入背包,任务完成自动发金币奖励并解锁成就。
4. 回去找艾拉 —— 此时对话里出现**条件门控**的新选项「水晶都找齐了!」。
5. 对话打开时移动被输入优先级层屏蔽,Esc 关闭对话。

## 代码结构(读这几个文件就够了)

| 文件 | 演示内容 |
|---|---|
| `src/game/content.ts` | 纯内容数据:NPC/对话树/任务/物品/成就,行为全部是声明式引用 |
| `src/game/engines.ts` | 装配点:创建注册表与引擎、注册效果/条件、事件总线接线 |
| `src/game/gold.ts` | 游戏专属系统如何通过效果注册表被内容引用 |
| `src/World.tsx` | SceneShell + Player(键盘+摇杆双输入)+ 昼夜灯光 + 小地图标记 + 逐帧拾取 |
| `src/ui/*` | 无头引擎的 UI 渲染:任务追踪、对话框、Toast、背包、小地图、虚拟摇杆 |

开发模式下 `window.__game` 暴露了引擎句柄与 `advance()`(后台标签页 RAF 暂停时手动驱动帧),
供调试与自动化测试使用。
